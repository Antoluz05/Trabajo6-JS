3) ¿Como es el manejo de las variables, tipos de datos y expresiones regulares de JS?

El manejo de las varibles se declara usando ver, let y const su manejo es que var tiene un alcance de funcion mientras que const y let tienen un alcance de bloque. Las variables const no pueden resignarse como las variables let que si pueden ser resignadas. Los tipos de datos son datos que no son objetos y no tienen ningun metodo y las expresiones regulares se manejan con los metodos RegExp test test() y exec() y con los métodos de String , match() , replace() , search() y split().
